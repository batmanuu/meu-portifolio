# Portfólio — Next.js + Tailwind CSS

Template de portfólio moderno com tema preto, rosa e branco, gradientes suaves e seções: Home, Sobre Mim, Habilidades, Projetos, Formação, Certificações, Contato.

Pré-requisitos: Node.js (>=16)

Instalação:

```bash
npm install
```

Executar em modo de desenvolvimento:

```bash
npm run dev
```

Compilar para produção:

```bash
npm run build
npm start
```

Personalize o conteúdo em `pages/index.js` e componentes em `components/`.
